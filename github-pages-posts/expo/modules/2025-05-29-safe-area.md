> 주의1 : `react-native`의 SafeAreaView가 아닌, `react-native-safe-area-context`의 SafeAreaView를 사용한다.
> 주의2 : `<SafeAreaProvider>`로 `<SafeArea>`를 감싸줘야한다.

#### 패키지 설치

```bash
npx expo install react-native-safe-area-context
```

#### 코드

```tsx
<SafeAreaProvider>
  <SafeAreaView
    style={{
      flex: 1,
    }}
  >
    {children}
  </SafeAreaView>
</SafeAreaProvider>
```
